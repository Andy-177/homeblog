# 如何在NDM下载器里使用油小猴插件解析的链接

我们都知道有一个神奇的油猴插件叫油小猴，它可以帮我们获取百度网盘的下载链接，但是这个链接无法直接使用，如果用浏览器和普通下载器下载会报错，那么接下来我就教大家修复这个问题。

这里主要教大家NDM下载器的方法，其他下载器也差不多。

# 教程

打开NDM，点`设置`，找到`默认用户代理`，修改默认用户代理也就是下载器的UA为下面的内容：

`pan.baidu.com`

修改完之后要勾选下面的`使用此UA用于手动下载和浏览器发送（请小心）`像下面这样即可:

- [x] 使用此UA用于手动下载和浏览器发送（请小心）

之后点击`确定`即可开始下载了

注意下载完成后重置UA并关闭`使用此UA用于手动下载和浏览器发送（请小心）`
