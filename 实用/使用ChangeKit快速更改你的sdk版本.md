# 使用ChangeKit快速更改你的sdk版本
是否还在为编译软件要使用不同版本Java或C#感到烦恼，是否感觉修改环境变量非常麻烦，使用ChangeKit解决你的这些烦恼。

ChangeKit是我开发的一套工具，选择好对应编程语言的软件，下载下来并放到不同版本sdk的根目录，只需复制它并创建快捷方式到桌面，即可快速切换开发环境。

这里我强烈建议不要使用安装程序而是直接使用二进制文件，因为二进制文件可以随意切换位置，而且不会搞乱你的环境变量，如果使用安装程序，那么他会直接在你的path里创建对应文件夹的路径。这可能会导致后面切换版本时没效果。

## 使用教程
1.下载对应二进制文件
2.把所有版本放在一个文件夹（同种编程语言）
3.创建对应的环境变量，比如Java就创建`JAVA_HOME`在系统变量的根下面，之后在`Path`变量里创建`%变量名%`，如果是Java则`Path`里创建`%JAVA_HOME%
3.下载对应编译器的Change软件，如Java下载ChangeJava，放到不同版本sdk的文件夹下，并在桌面创建Change对应的快捷方式
