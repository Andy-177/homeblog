# 通过服务器扫描仪扫描同一地址下哪些端口存在服务器
# 准备工作
1.[下载Server-Scanner的发布版，点击这里下载，软件作者B站UP小大小深](https://github.com/Andy-177/Server-Scanner/releases/download/Release-1.0.0/2.1.4.exe)

> [如果上面的无法下载，请点击这个链接跳转到加速器，使用加速器下载，并在输入框内输入下面的网址](https://gh-proxy.com/)

`https://github.com/Andy-177/Server-Scanner/releases/download/Release-1.0.0/2.1.4.exe`

2.你想扫描的服务器IP(`例如192.168.1.114`)或服务器域名(`例如abcd.fun`)
# 使用教程
1.新建一个文件夹，把下载的exe文件拖入文件夹中

2.双击exe，打开服务器扫描软件

3.`IP输入`一栏输入你想扫描端口的IP或域名

4.`线程数`一栏决斗扫描的质量和快慢，建议128

5.点击开始扫描后即可扫描整个域名下的服务器
## 提示
1.双击服务器的扫描结果会显示服务器的详细信息

2.点击`保存`可以保存你扫描的数据，使用`导入`即可轻松查看
